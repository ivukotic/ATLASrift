ATLASrift ATLASrift is a Virtual Reality application that provides an interactive, immersive visit to the ATLAS experiment. Using it, one can learn about the experiment as a whole, inspect individual sub-detectors, view real interactions, or take a scripted walkthrough explaining questions physicists are trying to answer.
